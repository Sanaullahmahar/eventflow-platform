# Events.com Frontend

React + Vite frontend for the Events.com marketing and event insurance experience.

## Structure

- `src/app`: app shell, providers, and route configuration
- `src/pages`: page-level entry components
- `src/features`: feature-scoped UI such as home page sections
- `src/shared`: shared layout, navigation, and configuration models
- `src/components/ui`: reusable UI primitives

## Scripts

- `npm run dev`: start the development server
- `npm run build`: create a production build
- `npm run test`: run Vitest tests
- `npm run lint`: run ESLint
